# SecGuard - 漏洞管理与跟踪平台

## 项目简介
基于 OWASP BLT 二次开发，打造轻量化、自动化、可协作的校园小型团队漏洞管理与跟踪平台。

## 技术栈
- 前端：React + Next.js + Tailwind CSS
- 后端：Python + Django + Django Ninja
- 数据库：PostgreSQL
- 缓存：Redis
- 容器化：Docker + Docker Compose
